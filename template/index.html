FocusTrap

This is a simple script used to trap a users focus within a specific HTMLElement.

  1. No Focus
  2. Focus
  3. Focus Inline Anchor
  4. Focus Anchor
  5. Focus Input
  6. Focus Input
  7. Input Disabled: No Focus
  10. Focus

Controls

Enable/Disable FocusTrap
Disable FocusTrap by pressing "Escape"
When enabled, autofocus to first element
When pressing tab, loop focus to beginning of focustrap
