FocusTrap

This is a simple script used to trap a user's focus within a specific HTMLElement.

Controls

Enable / Disable FocusTrap
Disable FocusTrap by pressing "Escape"
When enabled, autofocus to first element
Blur active element on disable
Stop at last element instead of looping
